# 🚗 AutoVault 3D – Premium Interactive Car Showroom

AutoVault 3D is a modern and interactive **3D Car Showroom** built using **HTML5, CSS3, JavaScript, and Three.js**. The project provides a realistic virtual showroom where users can explore premium cars in a 3D environment, customize vehicle appearance, compare different models, save favorite cars, and book test drives through an elegant and responsive interface.

The application focuses on delivering an immersive user experience with smooth animations, multiple camera angles, interactive controls, advanced filtering, and a clean futuristic UI.

## ✨ Features

* 🚗 Interactive 3D car visualization
* 🎥 Multiple camera views (Front, Side, Rear, Top & Default)
* 🎨 Car customization with multiple exterior colors
* 🛞 Wheel style customization
* 🪑 Interior theme customization
* ❤️ Wishlist system
* ⚖️ Compare multiple cars side by side
* 🔍 Search and filter cars by brand, body type, fuel type, and price
* 📊 Detailed specifications for every vehicle
* 💡 Interactive controls for headlights, doors, bonnet, and trunk
* 📅 Test drive booking form
* 🌙 Modern premium UI with responsive design
* ⚡ Smooth animations and loading screen
* 📱 Mobile-friendly interface

* Switch between different camera perspectives.
* View complete car specifications and features.

* Search and filter vehicles instantly.
* Book a test drive through the integrated booking form.

## 🎯 Purpose

The main objective of AutoVault 3D is to demonstrate modern front-end development and 3D web visualization by creating a realistic virtual car showroom experience. The project showcases responsive UI design, JavaScript interactivity, and Three.js-based 3D rendering while providing users with an engaging way to explore and compare automobiles online.
